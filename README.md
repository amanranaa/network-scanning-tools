# 🔍 Network Scanning Automation Toolkit

## 📌 Description
This project automates network scanning using Python and system tools:
- Ping (host discovery)
- ARP (network mapping)
- Nmap (port & service scanning)

## 🛠️ Installation

### Install Python
python --version

### Install Nmap
Linux:
sudo apt install nmap

Windows:
https://nmap.org/download.html

## ▶️ Usage

### Ping Scanner
python ping_scanner.py

### ARP Scanner
python arp_scanner.py

### Nmap Scanner
python nmap_scanner.py
